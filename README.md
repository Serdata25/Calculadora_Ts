Una calculadora web simple y funcional construida con TypeScript, HTML y CSS.

Características
Operaciones Fundamentales: Suma (+), Resta (-), Multiplicación (*), y División (/).

Funciones Avanzadas: Raíz cuadrada (2√x), Potencia al cuadrado (x²), y Recíproco (1/x).

Control de Input: Manejo de números decimales y cambio de signo (+/−).

Prevención de Errores: Manejo de la división por cero.

Interfaz Responsiva: Diseño sencillo y limpio con HTML/CSS.

🛠️ Tecnologías Utilizadas
TypeScript: Lenguaje principal para la lógica de la calculadora y tipado estático.

HTML5: Estructura de la aplicación.

CSS3: Estilos básicos de la calculadora.

Vite (o tu bundler/entorno): Utilizado para la configuración del proyecto y el entorno de desarrollo (si aplica).

🚀 Instalación y Ejecución Local
Sigue estos pasos para poner en marcha el proyecto en tu máquina local.

Prerrequisitos
Asegúrate de tener instalado Node.js (que incluye npm).

Pasos
Clonar el repositorio:

Bash

git clone https://www.youtube.com/watch?v=eQMcIGVc8N0
cd calculadora
Instalar dependencias:

Bash
npm install

Compilar y Ejecutar:
Inicia el servidor de desarrollo. (Ajusta este comando según tu package.json, si usaste Vite será similar a esto):

Bash
npm run dev
Abrir en el navegador:

El proyecto estará disponible en http://localhost:5173/ (o el puerto que te indique la consola).
